		  Afterglow color scheme    by Luke

Afterglow, a color scheme with a unique touch.

==============================================================================
Color scheme options					afterglow-options*

						*g:afterglow_transp_bg*
Set to 1 if you want a transparent background. Takes effect only in the
terminal.
>
	let g:afterglow_transp_bg = 0
<
